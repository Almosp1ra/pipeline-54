## 项目简介
本项目实现了一个支持 54 条 MIPS 指令的动态流水线 CPU。通过五个典型的流水线阶段（IF、ID、EX、MEM、WB）实现高性能指令执行，并支持中断处理机制。在下板过程中，可通过硬件按键/拨动开关触发中断，并在数码管上动态显示运算结果。

本项目仅包含 Verilog 源代码。

\* 虽然文件夹的名称为 pipeline-31.srcs，但是实际的 CPU 支持 54 条 MIPS 指令。名称不一致是因为最开始计划写 31 条指令CPU，但是设计的时候上头了干脆把剩下 23 条都支持了_(:з」∠)_

## 硬件与软件环境
* **操作系统**: Windows 11
* **开发环境**: Vivado 2016.2
* **硬件平台**: Xilinx NEXYS4 DDR

## 流水线结构
系统流水线由以下五个阶段组成：
1. **IF (Instruction Fetch)**: 取指，负责从 IMEM 获取指令并计算 PC+4。
2. **ID (Instruction Decode)**: 译码，生成控制信号，读取寄存器，进行立即数符号扩展。
3. **EX (Execute)**: 执行，处理算术逻辑运算及有效地址计算。
4. **MEM (Memory Access)**: 访存，执行存储器读写操作。
5. **WB (Write Back)**: 写回，将运算结果写回寄存器堆。

## 主要特性
- **指令集**: 支持 54 条标准 MIPS 指令。
- **中断支持**: 具备响应外部触发中断并恢复运行的能力。
- **可视化**: 支持通过 Nexys4 开发板上的数码管动态显示实时运算数值。
